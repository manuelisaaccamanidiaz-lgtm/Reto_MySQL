# Reto MySQL

## Descripción

Este repositorio contiene el desarrollo de diferentes retos y ejercicios prácticos realizados durante el proceso de aprendizaje de **MySQL**. Su propósito es reforzar los conceptos fundamentales del lenguaje SQL mediante la creación de bases de datos, manipulación de datos y consultas.

## Contenido

Los ejercicios están organizados en archivos según el tipo de instrucciones SQL:

* **DDL (Data Definition Language):** Creación de bases de datos, tablas y estructuras.
* **DML (Data Manipulation Language):** Inserción, actualización y eliminación de registros.
* **DQL (Data Query Language):** Consultas y transformación de datos mediante funciones SQL.

## Temas practicados

* Creación y eliminación de bases de datos.
* Creación de tablas.
* Tipos de datos.
* Llaves primarias y foráneas.
* Inserción, actualización y eliminación de registros.
* Consultas con `SELECT`.
* Filtros con `WHERE` y `HAVING`.
* Agrupación con `GROUP BY`.
* Ordenamiento con `ORDER BY`.
* Funciones de agregación (`SUM`, `COUNT`, `AVG`, `MIN`, `MAX`).
* Funciones de texto (`CONCAT`, `LENGTH`, `UPPER`, `LOWER`, `SUBSTRING`, `SUBSTRING_INDEX`).
* Funciones de fecha y lógica (`DATE_FORMAT`, `IF`).
* Vistas (`VIEW`).
* Índices (`INDEX`).
* Relaciones mediante `JOIN`.

## Tecnologías utilizadas

* MySQL
* MySQL Workbench
